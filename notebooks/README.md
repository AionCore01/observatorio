# Cuadernos de Laboratorio

> *"Los notebooks no son el laboratorio. Son los cuadernos donde nació el Observatorio."*

## Convención de nomenclatura

Cada notebook sigue el formato:

```
NNN_nombre_descriptivo.ipynb
```

Donde `NNN` es un número secuencial de 3 dígitos.

## Catálogo

| # | Nombre | Descripción | Fecha |
|---|--------|-------------|-------|
| 001 | `001_kaprekar_calendar.ipynb` | Primer experimento: la órbita de Kaprekar sobre el calendario 2024 | 2026-07-15 |
| 002 | `002_temporal_field.ipynb` | Campo temporal: distribución de pasos de convergencia | 2026-07-15 |
| 003 | `003_base_sweep.ipynb` | Barrido de bases 2-16: universalidad del fenómeno | 2026-07-16 |

## Reglas

- Cada notebook debe poder re-ejecutarse en un entorno limpio.
- Las hipótesis exploradas en un notebook deben registrarse en `LAB_NOTEBOOK.md`.
- Los notebooks son cuadernos de laboratorio, no el laboratorio.
